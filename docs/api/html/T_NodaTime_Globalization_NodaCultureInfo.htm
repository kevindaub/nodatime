NodaCultureInfo Class
Namespaces ► NodaTime.Globalization ► NodaCultureInfo

[This is preliminary documentation and is subject to change.]

Provides wrapper around a CultureInfo that supports NodaTime formatting information. This allows for the NodaFormatInfo data to be set into the CurrentCulture so it does not have to be passed around. If the underlying culture info is read only then this is also read only. To change it you need to clone it first.
