NodaTime.TimeZones Namespace
Send Feedback

[This is preliminary documentation and is subject to change.]

The NodaTime.TimeZones namespace contains types related to time zones beyond the core DateTimeZone class in the NodaTime namespace. Most users will have no need to refer to the types in this namespace.

Classes

  ClassDescription
BclTimeZone
Representation of a time zone converted from a TimeZoneInfo from the Base Class Library.
BclTimeZoneProvider
An IDateTimeZoneProvider implementation which uses TimeZoneInfo from .NET 3.5 and later.
EmptyDateTimeZoneProvider
Time zone provider which never provides any time zones. Setting this as the provider for the system will effectively mean that only UTC is recognized.
TzdbTimeZoneProvider
Provides an implementation of a IDateTimeZoneProvider that looks for its time zone definitions from a named resource in an assembly.
ZoneInterval
Represents a range of time for which a particular Offset applies.
ZoneLocalMapping
Complete information about the mapping from a local date and time within a time zone. This is usually obtained from a call to MapLocalDateTime(LocalDateTime)

Interfaces

  InterfaceDescription
IDateTimeZoneProvider
Provides the interface for objects that can retrieve time zone definitions given and id.

Enumerations

  EnumerationDescription
ZoneLocalMapping..::..ResultType
The type of result represented by this mapping. The underlying integer value is the same as the number of ZonedDateTime values which map to the original LocalDateTime.
