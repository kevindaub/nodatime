TryParse Method (value, formatProvider, result)
Namespaces ► NodaTime ► LocalTime ► TryParse(String, IFormatProvider, LocalTime%)

[This is preliminary documentation and is subject to change.]

Attempts to parse the given string using the specified format provider. If the parse is successful, the result is stored in the result parameter and the return value is true; otherwise Midnight is stored in the parameter and the return value is false.
