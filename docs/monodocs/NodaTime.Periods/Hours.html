Hours Class

An immutable time period representing a number of hours.

Example
Hours
is an immutable period that can only store hours. It does not store years, months or minutes for example. As such it is a type-safe way of representing a number of hours in an application.

The number of hours is set in the constructor, and may be queried using
Example
Value
. Basic mathematical operations are provided -
Example
Add()
,
Example
Subtract()
,
Example
Multiply()
and
Example
Divide()
.

Example
Hours
is thread-safe and immutable.

public sealed class Hours : SingleFieldPeriodBase, IComparable<Hours>, IEquatable<Hours>
