Seconds Class

An immutable time period representing a number of seconds.

Example
Seconds
is an immutable period that can only store seconds. It does not store years, months or hours for example. As such it is a type-safe way of representing a number of seconds in an application.

The number of seconds is set in the constructor, and may be queried using
Example
Value
. Basic mathematical operations are provided -
Example
Add()
,
Example
Subtract()
,
Example
Multiply()
and
Example
Divide()
.

Example
Seconds
is thread-safe and immutable.

public sealed class Seconds : SingleFieldPeriodBase, IComparable<Seconds>, IEquatable<Seconds>

ToStandardDays () : Days
This method allows you to convert between different types of period. However to achieve this it makes the assumption that all days are 24 hours long, all hours are 60 minutes long and all minutes are 60 seconds long. This is not true when daylight savings time is considered, and may also not be true for some unusual chronologies. However, it is included as it is a useful operation for many applications and business rules.

⊟ ToStandardDays Method

This method allows you to convert between different types of period. However to achieve this it makes the assumption that all days are 24 hours long, all hours are 60 minutes long and all minutes are 60 seconds long. This is not true when daylight savings time is considered, and may also not be true for some unusual chronologies. However, it is included as it is a useful operation for many applications and business rules.
