Minutes Class

An immutable time period representing a number of minutes.

Example
Minutes
is an immutable period that can only store minutes. It does not store years, months or hours for example. As such it is a type-safe way of representing a number of minutes in an application.

The number of minutes is set in the constructor, and may be queried using
Example
Value
. Basic mathematical operations are provided -
Example
Add()
,
Example
Subtract()
,
Example
Multiply()
and
Example
Divide()
.

Example
Minutes
is thread-safe and immutable.

public sealed class Minutes : SingleFieldPeriodBase, IComparable<Minutes>, IEquatable<Minutes>
