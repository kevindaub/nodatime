public NodaTime.DateTimeZone ForId (string id)

⊟ Parameters

id
The id of the time zone to return.

⊟ Returns

The NodaTime.DateTimeZone or null if there is no time zone with the given id.

⊟ Remarks

If the time zone does not yet exist, its definition is loaded from where ever this provider gets time zone definitions. Time zones should not be cached in the provider as they will be cached in NodaTime.TimeZones.DateTimeZones.
