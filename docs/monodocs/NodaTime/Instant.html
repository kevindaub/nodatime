Instant Struct

Represents an instant on the timeline, measured in ticks from the Unix epoch, which is typically described as January 1st 1970, midnight, UTC (ISO calendar). (There are 10,000 ticks in a millisecond.)

public struct Instant : IComparable<Instant>, IEquatable<Instant>

⊟ Remarks

This type is immutable and thread-safe.
