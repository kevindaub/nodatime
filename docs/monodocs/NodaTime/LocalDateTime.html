LocalDateTime Struct

A date and time in a particular calendar system.

public struct LocalDateTime : IEquatable<LocalDateTime>

⊟ Remarks

A LocalDateTime value does not represent an instant on the time line, mostly because it has no associated time zone: "November 12th 2009 7pm, ISO calendar" occurred at different instants for different people around the world.

This type defaults to using the IsoCalendarSystem unless a different calendar system is specified.

This type is immutable and thread-safe.
