Interface describing a time zone.

public abstract class DateTimeZone

⊟ GetName Method

Returns the name associated with the given instant.

public virtual string GetName (Instant instant)

⊟ Parameters

instant
The instant to get the name for.

⊟ Returns

The name of this time. Never returns null.

⊟ Remarks

For a fixed time zone this will always return the same value but for a time zone that honors daylight savings this will return a different name depending on the time of year it represents. For example in the Pacific Standard Time (UTC-8) it will return either PST or PDT depending on the time of year.

⊟ SystemDefault Property

Gets the system default time zone which can only be changed by the system.

public static DateTimeZone SystemDefault { get; }

⊟ Value

The system default NodaTime.DateTimeZone. this will never be null.

⊟ Remarks

The time zones defined in the operating system are different than the ones defines in this library so a mapping will occur. If an exact mapping can be made then that will be used otherwise UTC will be used.
