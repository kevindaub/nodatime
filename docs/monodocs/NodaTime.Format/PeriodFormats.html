PeriodFormats Class

Provides access to constructed instances of PeriodFormatter.

public static class PeriodFormats

⊟ Remarks

Period formatting is performed by the NodaTime.Format.PeriodFormatter class. Three classes provide factory methods to create formatters, and this is one. The others are NodaTime.Format.PeriodFormatterBuilder and NodaTime.Format.IsoPeriodFormats.

PeriodFormats is thread-safe and immutable, and the formatters it returns are as well.
