⊟ Parse Method

Parses a datetime from the given text, returning a new ZonedDateTime.

public NodaTime.ZonedDateTime Parse (string text)

⊟ Parameters

text
The text to parse

⊟ Returns

Parsed value in a ZonedDateTime object

⊟ Exceptions

Type Reason
NotSupportedException If parsing is not supported
ArgumentException If the text to parse is invalid

⊟ Remarks

If the text contains a time zone string then that will be taken into account in adjusting the time of day as follows. If the DateTimeFormatter.WithOffsetParsed has been called, then the resulting ZonedDateTime will have a fixed offset based on the parsed time zone. Otherwise the resulting ZonedDateTime will have the zone of this formatter, but the parsed zone may have caused the time to be adjusted.

⊟ WithCalendar Method

Returns a new formatter that will use the specified calendar system in preference to that of the printed object, or ISO on a parse.

public DateTimeFormatter WithCalendar (NodaTime.Calendars.CalendarSystem newCalendarSystem)

⊟ Parameters

newCalendarSystem
The calendar system to use as an override

⊟ Returns

The new formatter

⊟ Remarks

When printing, this calendar system will be used in preference to the calendar system from the datetime that would otherwise be used.

When parsing, this calendar system will be set on the parsed datetime.

A null calendar system means no-override.
