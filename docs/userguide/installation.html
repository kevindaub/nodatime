Installation

Our primary distribution channel is NuGet with two packages:

  • NodaTime
  • NodaTime.Testing

System requirements

Currently Noda Time requires .NET 3.5 (client profile). It supports Mono (with some caveats) but does not support Silverlight (or, by extension, Windows Phone 7). We'd like to support these in the future, but it will involve a non-trivial amount of work.
